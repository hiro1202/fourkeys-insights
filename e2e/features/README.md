# features/

このディレクトリでは **Gherkin 記法** でテストケースを管理しています。

- 公式: <https://cucumber.io/docs/gherkin/reference/>
- 日本語キーワード対応表: <https://cucumber.io/docs/gherkin/languages/>

## 日本語キーワード早見表

ファイル先頭に `# language: ja` を書くと以下のキーワードが使えます。

| 英語 | 日本語 |
|---|---|
| Feature | `機能` |
| Scenario | `シナリオ` |
| Scenario Outline | `シナリオアウトライン` |
| Background | `背景` |
| Given | `前提` |
| When | `もし` |
| Then | `ならば` |
| And | `かつ` |
| But | `しかし` |
| Examples | `例` |
| Rule | `ルール` |
